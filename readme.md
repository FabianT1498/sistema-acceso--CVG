<h2 align="center">Sistema de control de acceso de visitantes</h2>
<p>El sistema COINVI tiene como objetivo controlar el acceso de los visitantes de los edificios sedes de la Corporación Venezolana de Guayana (CVG). El sistema esta dotado de las siguientes funcionalidades</p>

* Gestion de usuarios con distintos roles
* Gestion de visitantes
* Gestion de autos
* Gestion de visitas y reportes
* Carga de imagenes
* Generación de pases en formato PDF

<p>Tecnologias utilizadas:</p>

Tecnologia  | Version
------------- | -------------
PHP  | 7.1.3
Laravel  | 5.8.*
Jquery | 3.4.1
